== Description ==

# RenewAI Featured Image Generator

RenewAI Featured Image Generator is a powerful WordPress plugin that leverages AI technology to automatically generate featured images for your posts. By integrating OpenAI for prompt generation and advanced image generation models, this plugin streamlines your content creation process.

## Free Version Features:
* AI-powered prompt generation using GPT-3.5 Turbo
* High-quality image generation using DALL-E 3
* Basic image size options
* Easy-to-use interface integrated into the post editor
* Secure API key management

## Premium Version Features:
* Everything in the free version, plus:
* Access to Flux image generation models:
  - Flux Pro 1.1 (Latest version with improved quality)
  - Flux Pro (Stable production version)
  - Flux Dev (Development version)
* Advanced image size options
* Choice of GPT models (GPT-4, GPT-4-32k)
* Additional image styles (Natural, Artistic, Realistic, Abstract)
* Priority support

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> RenewAI Image screen to configure the plugin and add your API keys.
4. Visit the Help page under RenewAI Image for detailed setup instructions and troubleshooting tips.
5. Language files can be found in the 'languages' directory for translation.

== Frequently Asked Questions ==

= What's the difference between free and premium versions? =

The free version includes DALL-E 3 for image generation and GPT-3.5 Turbo for prompt generation. The premium version adds access to Flux image generation models, advanced GPT models, and additional customization options.

= Do I need API keys to use this plugin? =

Yes. For the free version, you need an OpenAI API key. For the premium version, you'll need both OpenAI and Flux API keys.

= How much does it cost to use the API services? =

* Free version: OpenAI API usage costs apply (see OpenAI pricing page)
* Premium version: Both OpenAI and Flux API usage costs apply (see respective pricing pages)

= Is there a limit to how many images I can generate? =

The number of images you can generate depends on your subscription plans with OpenAI and Flux. Please refer to their pricing pages for more information.

= Can I translate this plugin into my language? =

Yes! The plugin is fully translation-ready. All strings are properly internationalized using WordPress standards. You can translate it using any WordPress translation tool.

= What languages are supported? =

The plugin supports all languages that WordPress supports. The text domain 'renewai-featured-image-generator' is used for translations.

= Does this plugin support RTL languages? =

Yes, the plugin fully supports RTL (Right-to-Left) languages and properly displays all interface elements in RTL mode.

= Compatible with WordPress language switchers =

Yes, the plugin is compatible with WordPress language switchers.
